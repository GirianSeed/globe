Sample program for manipulating asynchronous files

<Description>
This sample program shows how to asynchronously copy a file on the host to the hard disk.

<File>
	main.c

<Run Method>
	% make:  Compile
	% make run:  Execute

The following method can also be used to run the program after it is compiled.
	% dsedb
	> run main.elf

The program has terminated normally when this is displayed: 
	sample end
