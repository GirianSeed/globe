Asynchronous File Operation Sample


<Description of the Sample>
	This is a sample for copying files in the host to hard disks
	asynchronously.

<File>
	main.c

<Activating the program>
	% make		: Complies the program
	% make run	: Executes the program

	After the compilation, the program is also executable with the
	following:

	% dsedb
	> run main.elf

	Execute a normal termination if the following is displayed.

	sample end.
	

<Using the Controller>
	None
