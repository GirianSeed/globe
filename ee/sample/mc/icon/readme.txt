[3D icon tool]

This directory contains two Linux tools that are used for 3D icon creation.


* 3D icon file ASCII/binary conversion program
-----------------------------------------------
This tool allows the contents of an existing icon file to be viewed in text format. Also, the text output can be edited to modify the contents of the icon file.
        sce/ee/sample/mc/basic/slime[1-3].ico
are provided as sample icon files.


* 3D memory card icon texture compression program
--------------------------------------------------
This program compresses icon textures in formats supported for icon files.


Please refer to the Memory Card library overview for details about the 3D icon format and the icon.sys format used to specify icons that are displayed on the main unit's memory card control screen.
